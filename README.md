## Qué encontrarás aquí

Los writeups siguen, en mayor o menor medida, este flujo:

1. Reconocimiento
2. Enumeración
3. Explotación
4. Escalada de privilegios
5. Conclusión

## Front matter y metadatos

Todos los writeups siguen un esquema de front matter homogéneo con estos campos:

- estado
- plataforma
- sistema operativo
- dificultad
- vector inicial
- servicio inicial
- puerto inicial
- credenciales
- usuarios
- técnica de escalada
- técnicas utilizadas
- herramientas utilizadas
- fecha

## Nota

Este repositorio está orientado a documentación técnica y aprendizaje. La idea es explicar el proceso real de resolución, no amontonar comandos sin contexto como si eso fuera una metodología.

## Autor

Si algún writeup te resulta útil, puedes invitarme a un café desde los enlaces incluidos en algunos laboratorios.
